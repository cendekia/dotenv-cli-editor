## Install

```
$ npm i -g dotenv-cli-editor
```

## Usage

```
$ envi --help

  Usage:  <command>

  Options:
    -v, --version            output the version number
    -h, --help               output usage information

  Commands:
    edit [options] <envKey>  Edit specific env parameter value
```
